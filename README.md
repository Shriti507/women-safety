# Women Safety App

A mobile application built with **React Native (Expo)** to enhance personal safety through real-time location sharing, emergency alerts, video recording, and quick-access safety features. The app provides fast, reliable tools designed to support users in unsafe situations.



## Demo Link
https://drive.google.com/file/d/1MwyJ4H_GLfA1fs4xpu498qNAMsAOIA-9/view


## Tech Stack

- React Native (Expo)
- Expo Camera
- Expo Location
- Expo MediaLibrary
- React Navigation
- AsyncStorage
- Linking APIs (SMS, Call, WhatsApp)

---

## Installation and App Initiation

Follow these steps to set up and run the project locally.

### 1. Clone the Repository
```bash
git clone https://github.com/Shriti507/women-safety.git
cd women-safety
cd frontend
cd safety-app
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Development Server
```bash 
npx expo start
```
